Introduction

What is HTML?

HTML stands for Hyper Text Markup Language. This language is the standard markup language for creating Web pages.
HTML describes the structure of a web page. By using a series of elements, HTML tells the browser how to display the content.
HTML, as a markup language uses tags to identify content by using labels such as "this is a heading", "this is a paragraph", "this is a link", etc.

HTML Anatomy

HTML element

Let's explore the paragraph element further:

<p>Hello world</p>

  • The opening tag ("<p>"): this consists of the name of the element (in this case p), wrapped in opening and closing angle brackets. Ths states where the element begins or starts to take effect -- in this case, the start of the paragraph
  • The closing tag ("</p>"): is the same as the opening tag, but it inclues a forward slash ("/") before the element's name. This states where the element ends -- in this case, it ends where the paragraphs ends.
  • The content ("Hello world"): this is the content of the element, which in this case is just text.
  • The element ("<p>Hello world</p>"): The opening tag, the closing one and the contact together compromise the element.

    • <p class="id">Hello world</p>

    Elements can also have attributes. Attributes bring extra information about the element that you don't want to apeear in the actual content.
    Here, class is the attribute name and "id" is the value.
    There are multiple attributes, according to the base element that is being used in HTML.

    HTML documents

    An HTML element is defined by a start tag, some content and an end tag. Here are some examples:

    <!DOCTYPE html>
    <html>
    <head>
    <title>Page Title</title>
    </head>
    <body>
    <h1>My first heading</h1>
    <p>My first paragraph.</p>
    </body>
    </html>

    Example explained

  • The <!DOCTYPE html> declaration defines that this is an HTML document
  • The <html> element is the root element of an HTML page
  • The <head> element contains the meta information about the HTML page
  • The <title> element specifies a title for the HTML page (which is shown in the browser's title bar or in the page's tab)
  • The <body> element defines the document's body, and is a container for all the visible contents, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.
  • The <h1> element defines a large heading
  • The <p> element defines a paragraph

    • Nesting Elements

    You can put elelemnts inside other elements too -- is called nesting. If we wanted to give some emphasis to our "Hello World" statement, for example Hello World, we would wrap the whole element in a <strong> element.

    <p> <strong>Hello World</strong></p>

    However, you do need to make sure that your elements are properly nested. In the example above, first the <p> element was opened and then came the <strong> element; therefore, wee need to close first the </strong> element and then the </p> element

    <p><strong>Hello World</p></strong>

    The elements have to open and close correctly, so that they are clearly inside or outside one another. If they overlap as shown above, then your web browser will try to make the best guess at what you were trying to say, which can lead to unexpected results -- including not working.

    • HTML Attributes

    Elements in HTML have attributes; these are additional values that configure the elements or adjust their behavior in various ways to meet the criteria users want.

    Attributes usually come in name/value pairs like: name="value"

    Attribute name Elements Desctription

    href

    <a>

    The <a> tag defines a hyperlink. The href attribute specifies the URL of the page the link goes to.

    src

    <img>

    The <img> tag is used to embed an image in an HTML page. The src attribute specifies the path to the image to be displayed

    width, height

    <img>

    The <img> tag should also contain the width anf height attribuets, which specifies the width and height of the image.

    alt

    <img>

    The required alt attribute for the <img> tag specifies an alternate text for an image, if the image for some reason cannot be displayed. This can be due to a slow connection, an error on the src attribute or the usage of a screen reader.
